---

## TLDR

```bash
simdrive seats list                              # who's bound to your team key
simdrive seats add alice@acme.com                # provision a seat
simdrive seats revoke alice@acme.com             # release a seat
```

Each Team seat is independently machine-locked. Adding a seat issues a
per-seat sub-key; revoking releases it within minutes.
